TCP与UDP的区别

1 TCP是面向连接的传输控制协议 UDP提供无连接的数据包服务
2 TCP是可靠的数据传输协议  UDP不可考
3 TCP占用的系统资源比较多,UDP比较少.
4 UDP的实时性TCP高.
